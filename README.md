# Quick ML Demo

Este es un proyecto de demostración rápida de Machine Learning. El objetivo es crear un modelo simple que pueda ser utilizado para predecir valores basados en un conjunto de datos pequeño.

## Características

- **Modelo Simple**: Implementación de un modelo de regresión lineal.
- **Datos de Ejemplo**: Uso de un conjunto de datos sintético para entrenamiento y prueba.
- **Evaluación**: Métricas básicas de evaluación del modelo.

## Instalación

1. Clona el repositorio:
   ```bash
   git clone https://github.com/RanuK12/quick-ml-demo.git
   ```
2. Instala las dependencias:
   ```bash
   pip install -r requirements.txt
   ```

## Uso

Ejecuta el script principal para entrenar y evaluar el modelo:
   ```bash
   python main.py
   ```
